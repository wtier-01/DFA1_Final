# FDA1 Final Project — Factor Model and Time Series Analysis

## Project Overview

This project analyzes two U.S.-listed individual common stocks using time-series, regression, and factor-model tools from Financial Data Analytics I.

The central research question is:

> What drives the returns of these two stocks, and what is left unexplained?

The project compares:

| Ticker | Company | Sector |
|---|---|---|
| MSFT | Microsoft Corporation | Technology |
| LMT | Lockheed Martin Corporation | Industrials / Aerospace & Defense |

These stocks were selected because they are U.S.-listed common stocks, are not on the excluded ticker list, have long daily trading histories, and represent clearly different business profiles.
